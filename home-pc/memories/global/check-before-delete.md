# 删前先查，确认无害再动手

## 规则
删除任何文件、快捷方式、目录之前，**必须先检查它是什么**，确认无害再删。

## 触发场景
- 删快捷方式前 → 检查指向的目标路径
- 删文件前 → 检查文件类型/用途
- 删目录前 → 检查目录内容

## 反面案例
清理 D:\根 时，看到「扫描 (Desktop-o56g1ts) - 快捷方式.lnk」就删了，没检查是指向哪台电脑的共享文件夹。用户需要这个快捷方式，已成误删。

**Why:** 删前不查 = 可能删掉用户需要的东西
**How to apply:** 任何删除操作前，先查内容再动手
