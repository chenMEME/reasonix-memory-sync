# Reasonix Memory Sync

> ⚠️ **项目状态：实验性 / 开发中**
>
> 这是作者自用的多设备记忆同步方案，目前**尚不成熟**。API 可能会频繁变动，文档可能滞后于代码，
> 部分功能可能存在边界情况未覆盖。如果觉得不好用，那很正常——欢迎提 Issue 或 PR 一起改进，
> 但请勿期望这是一个开箱即用的稳定产品。

基于 Git 的多设备记忆同步开放标准。

## 快速开始

1. 克隆本仓库
2. 在电脑 A 上对你的 agent 说 **"上传记忆"**
3. 在电脑 B 上对你的 agent 说 **"更新记忆"**

系统会自动识别电脑、拉取远程记忆、比对合并、翻译路径差异。

## 标准规范

详见 [STANDARD.md](STANDARD.md) v1.0

## 功能

- ✅ N 台电脑任意数量
- ✅ 全局记忆同步
- ✅ 项目记忆同步（自动处理 hash 路径）
- ✅ Skills 版本管理
- ✅ 路径自动翻译（Windows \\ ↔ Linux /）
- ✅ 冲突自动合并（时间戳 + 版本号）
- ✅ 删除同步
- ✅ 配置互补
- ✅ 应用前自动备份
